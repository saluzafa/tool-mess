## index.html generation prompt
```text
Generate an index.html page that links every html tools present in the directory `tools`.
Links must appears has tiles with the tool name and a beautiful icon.
There can be no more than 6 tiles per row.
Create categories based on where the tools are located, for tools that do not have a directory create a "Miscellaneous" category.
The "Miscellaneous" category should be the last category.
Always exclude files/directories listed in @.gitignore
Use Tailwind V4 (CDN version) for the CSS/styling by using `<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>`.
```

## index.html update prompt
```text
Update the index.html page to include any new, updated or removed tools in the directory `tools`.
Links must appears has tiles with the tool name and a beautiful icon from Font-Awesome.
There can be no more than 6 tiles per row.
Create categories based on where the tools are located, for tools that do not have a directory create a "Miscellaneous" category.
The "Miscellaneous" category should be the last category.
Always exclude files/directories listed in @.gitignore
```
